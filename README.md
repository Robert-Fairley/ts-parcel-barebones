# ES6 Parcel Boilerplate

### Setup

You'll need a current version of NPM installed as well as Parcel:

https://www.npmjs.com/

https://parceljs.org/

Install dependencies:

```shell
npm install
```

### Run

To run a hot-reload development version:

```shell
npm start
```

### Build

To build for production:

```shell
npm run build
```

## Tests

To run unit tests:

```shell
npm test
```
